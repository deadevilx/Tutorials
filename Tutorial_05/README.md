#  Tutorial_05 - Data structures

    A data structure is a group of data elements grouped together under one name.
    e.g.
> struct Circle {

>    double center_x;

>    double center_y;

>    double radius;

> };

    or
    
> typedef struct {

>    Circle base;

>    double height;

> } Cone;

Please read this [page](https://cplusplus.com/doc/tutorial/structures/)
